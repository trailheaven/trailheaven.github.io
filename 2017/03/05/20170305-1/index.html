【堀江貴文】議員をボット化。「ホリエモン新党」の戦略（NEWS PICKS）

>>記事はこちら

われわれの党は、議員が全部ボットなんです。それを条件に議員になってもらう。要はクラウドファンディングで何千万円か集められれば、東京のすべての選挙区に候補者を出せます。候補者は当選したら、議員として、みんなの意見の総意に基づいて投票するわけです。
面白い発想だなぁ。良い悪いなんてないんだし、まずは実現してみたい。


プレミアムフライデー、オフィス街飲食店は「大迷惑」な理由（DIAMOND ONLINE）

>>記事はこちら

2月24日、月末の金曜日に早期退社を促す「プレミアムフライデー」がいよいよ始まる。旅行業界や外食業界などで個人消費が増えて、経済が活性化するとされているが、意外にもオフィス街の飲食店では、「大迷惑」という怒りや不安の声が大きい。
明るいうちから飲めるなんて贅沢だなぁ、て思うのは自分だけ？お店からすれば労もなくこの贅沢感が与えられるのに、ネガティヴ要素しか出てこないのが残念。この昂ぶる気持ちを使って、もっと日頃味わえないプレミアム感を演出してくれればいいのに。そうすれば客単価だって上がるのでは？

>>記事はこちら

結局のところ、この先も「AIに仕事を奪われる」のではなくて「人手が無いのでAIに仕事をしてもらう」なんじゃない？
